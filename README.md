# Soft Machine — Security

This repository is the **security disclosure channel** for
[Soft Machine](https://soft-machine.io), a Virtual Machine–based agentic
development environment / Cloud OS.

If you have found a security vulnerability in any Soft Machine asset — the
web application, the control-plane API, workspace microVMs, the port proxy, or
our supply chain — please report it to us privately and responsibly.

## 🛡️ Report a Vulnerability

**[➡️ Report a Vulnerability](https://github.com/Soft-Machine-io/security/security/advisories/new)**

All reports are handled privately through GitHub Security Advisories. Please do
**not** open public issues, pull requests, or discussions for security
problems — public issues on this repository are disabled for this reason.

Previously coordinated advisories are published at:
**https://github.com/Soft-Machine-io/security/security/advisories**

## 📋 Policy

Before reporting, please read our full policy:

- **[SECURITY.md](./SECURITY.md)** — scope (in and out), rules of engagement,
  legal **safe harbor**, response SLAs, and our coordinated-disclosure
  timeline (90 days by default).

Key commitments:

- **Acknowledgement within 72 hours**
- **Triage within 7 days**
- Good-faith research conducted under this policy is **authorized** and covered
  by [disclose.io](https://disclose.io)-standard safe harbor.
